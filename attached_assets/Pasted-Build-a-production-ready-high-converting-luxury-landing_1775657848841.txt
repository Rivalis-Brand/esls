* Site Conditions:

  * Easy
  * Moderate (+10%)
  * Difficult (+20%)

* Add-ons:

  * Pet infill (+$2/sq ft)
  * Drainage (+$3/sq ft)
  * Custom design (+$500)

---

### CALCULATION LOGIC:

1. Base = squareFootage × pricePerSqFt
2. Add add-ons (per sq ft + flat fees)
3. Apply multiplier:

   * Moderate → ×1.10
   * Difficult → ×1.20
4. If >1500 sq ft → apply 5% discount

## FIRESTORE RULES

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    match /inquiries/{docId} {
      allow create: if true;
      allow read, update, delete: if false;
    }

    match /estimates/{docId} {
      allow create: if true;
      allow read, update, delete: if false;
    }

    match /reviews/{docId} {
      allow read: if true;
      allow create, update, delete: if request.auth != null
        && request.auth.uid == "ADMIN_UID_HERE";
    }
  }
}
```
